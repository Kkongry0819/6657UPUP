| **这软件是干嘛的？** | 使用C# .NET和WebView2开发的专为直播间玩小将设计的“弹幕辅助工具”，支持海量烂梗秒拷及 AI 生成。 |
| **打不开怎么办？** | 请确保已安装 [WebView2 Runtime](https://developer.microsoft.com/en-us/microsoft-edge/webview2/)，Win11 自带，Win10 用户需手动安装。 |

| **不填 API Key 能用吗？** | 可以！基础的烂梗拉取和复制完全免费，无需 API Key。 |
| **没满15级可以发弹幕吗** | 可以！但是别人看不见，相当玩小将得斗鱼15级并且有玩大哥的粉丝牌。 |
